Open Source Shakespeare API

General notes

This API provides access to the works of Shakespeare, including plays, sonnets, and other works. The data is sourced from the public domain works of Shakespeare and is provided in a structured format for easy consumption by developers. Currently it only runs on localhost, we are looking into hosting this API.

API detail

Chapters

Access all the chapters of the works of Shakespeare.

/chapters

Get all

  • Curl
  • HTTP

Paragraphs

Access all the paragraphs of the works of Shakespeare.

/paragraphs

This is the endpoint you should use if you want to return the paragraph containing any words or phrases. Note you can include as many words or phrases as you like by adding extra terms. This matches by the whole word or phrase. You can also return all the paragraphs by workId or charId. All of these endpoint paragraphs can optionally include the details about the work by including the workInfo=true parameter, if workInfo is set to false , or isn’t included, the work info will not be included. It’s currently in beta and functionality will be extended in v2.

  • Curl
  • HTTP
curl -X GET "{{host}}/paragraphs"
GET %7B%7Bhost%7D%7D/paragraphs HTTP/1.1
Host:

/paragraphs/:paragraphId

Get paragraph by paragraphId

  • Curl
  • HTTP

GET %7B%7Bhost%7D%7D/paragraphs/859870 HTTP/1.1
Host:

Quotations

Get a list of quotations for the works of Shakespeare

/quotations

Get all quotations or get quotations by work. Note that work is slightly different here, it is not workId it is the name of the actual work i.e. “The Tempest”. It allows for upper or lower case searching

Works

Get a list of all of the works of Shakespeare.

/works

Get a list of all of the works, this can be sorted by genre, title as all or part of the title, or date as a year.

  • Curl
  • HTTP
curl -X GET "{{host}}/works?genre=tragedy&title=an&date=1606"
GET %7B%7Bhost%7D%7D/works?genre=tragedy&title=an&date=1606 HTTP/1.1
Host:

/works/:workId

Get work by work id. This takes the optional query params of chapters (returns the work with all the chapters) and/or paragraphs (returns all of the paragraphs of the work). If paragraphs and chapters are false or not present they will not be returned.

  • Curl
  • HTTP
curl -X GET "{{host}}/works/allswell"
GET %7B%7Bhost%7D%7D/works/allswell HTTP/1.1
Host:
